GELOS TASK MANAGER INSTRUCTIONS

  1. Go to the Task Manager to add new tasks.
  2. Fill in task name, note, due date and select priority.
  3. Click on Add Task to save the task.
  4. The saved task will be displayed below the form.
  5. The saved tasks can be marked as completed by ticking on a checkbox or deleted by clicking on delete button.
  6. Completed tasks will be shown with a strikethrough.
  7. If you need any assistance, please use the Contact.
